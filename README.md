# FrameKit

Add professional camera metadata frames to your iPhone photos.

## What it does

FrameKit transforms your photos by adding elegant white frames that display camera information like:
- Device model (e.g., "Shot on **iPhone 15 Pro**")
- Technical specs (focal length, aperture, shutter speed, ISO)

Perfect for photographers who want to showcase their work with a professional touch.

## Requirements

- iOS 26.0+
- Xcode 16.0+
- Swift 6.0+

## Tech Stack

- **SwiftUI** - Modern declarative UI
- **CoreGraphics** - High-quality image rendering
- **SwiftData** - Persistent storage
- **MVVM Architecture** - Clean, maintainable code

## Installation

1. Clone the repository
2. Open `FrameKit.xcodeproj` in Xcode
3. Build and run on your device or simulator

## Privacy

FrameKit requires photo library access to:
- Select photos to frame
- Save your framed creations

All processing happens locally on your device. No data is collected or shared.
